# Smart-Field-Survey-App

A React Native Expo app for field survey data collection with camera capture, photo attachment, and survey history management.

## Recent updates
- Captured photos can now be saved to the device gallery after taking them in the camera screen.
- Survey photos can also be attached to the current survey entry for later review.

## Getting started
1. Navigate to the app folder: `cd SmartFieldSurvey`
2. Install dependencies: `npm install`
3. Start the app: `npm start`
